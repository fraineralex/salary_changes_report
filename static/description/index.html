Salary Changes Report

Odoo module that allows you to export a report of salary changes for employees to Excel.

Open the wizard in the reports menu in the payroll module

Supports multiple Payslip Structures

The module exports detailed salary changes report to Excel, streamlining compliance and optimizing resource management.

Installation Note!

Before you install the module, You need to install this app first: the odoo hr.payroll base module and the python module xlwt
